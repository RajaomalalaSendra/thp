Mercredi

JavaScript, et pimper tes sites

Hier, tu as vu comment un site basique et statique avec HTML et CSS. Aujourd'hui, nous allons te montrer comment lui donner un peu de peps avec JavaScript, un langage de programmation conçu pour faire des animations de pages web côté client. Pour ceci nous allons continuer notre histoire des sites internet avec JavaScript, puis nous te montrerons comment utiliser la console de ton browser, qui permet de faire des trucs de hackeurs 😉

Aujourd'hui, tu vas apprendre JavaScript, un peu d'histoire, comment le brancher à sa page web, puis tu devras mettre en application ce qui tu as appris en donnant un peu d'animations aux pages que tu as construites hier avec HTML CSS.

Déroulé

1. JavaScript

1.1. Introduction

Pour les ressources aujourd'hui, nous allons te donner de quoi comprendre ce que fait JavaScript, la console de ton browser, puis nous allons te donner des ressources pour apprendre JavaScript.

1.2. Historique

Avant de savoir ce que fait JavaScript, nous allons te donner une leçon qui te permet de comprendre la différence entre le Client, et le Serveur. Voici une vidéo qui explique cela.

Ensuite, nous allons positionner JavaScript dans le contexte de HTML et CSS, et qu'est-ce que JavaScript apporte aux deux langages que tu as vu hier. Voici une vidéo qui fait ce récap.

Enfin, un peu d'histoire ! Nous allons te donner un cours rapide sur JavaScript, pourquoi il a été créé, qu'est-ce que JavaScript ? Ce chapitre du cours JavaScript de OpenClassrooms fait exactement cela ✌️

1.3. Le cours

1.3.1. La console

La console est un outil extrêmement puissant qui permet de jouer avec le code HTML / CSS reçu, vérifier ce que les fichiers t'envoient, les réponses du serveurs. Tu peux même manipuler les pages pour les changer de ton côté. Et c'est très pratique quand on développe du front.

Pour t'apprendre la console, nous allons prendre cette vidéo de José Vidal qui donne en 10 minutes un aperçu de ce qu'est que l'inspecteur d'éléments, et comment tu peux t'en servir.

1.3.2. Découverte de JavaScript

Ahh, JavaScript, quel beau langage de programmation. Ce cours sera en plusieurs parties. Dans la première, nous allons te demander de faire un peu de CodeCademy pour te mettre au fait sur la syntaxe de JavaScript. Ensuite, nous te demanderons de suivre un tutoriel pour brancher ta première page JavaScript. Et après tu sera paré pour t'aventurer dans le beau projet que nous t'avons réservé.

Nous allons te demander de commencer par faire le premier chapitre de la track JavaScript de CodeCademy. C'est une bonne introduction à comment déclarer des variables, des fonctions, etc. Learn Javascript: Introduction - Introduction to JavaScript

Une fois que tu as fait ceci, nous allons te demander de suivre un tutoriel et de faire le tutoriel d'utilisation de Javascript de la Fondation Mozilla, afin de créer ta première page dynamique \o/

Enfin, il ne te reste plus qu'à découvrir ce site merveilleux qu'est w3schools, notamment sur leur page d'exemples des fonctions en JavaScript qui permettent d'intéragir avec une page. Pour le moment, pas besoin d'apprendre par coeur tout ce qu'ils mettent, il te suffit juste de savoir réutiliser leur information, notamment au moment des projets.

1.4. Points importants à retenir

Voici les points importants à retenir :

  • La console de ton explorateur internet est un outil extrêmement puissant qui te permet d'intéragir facilement avec ta page
  • JavaScript est un langage qui te permet de dynamiser tes pages web

1.5. Pour aller plus loin

Pour aller plus loin, tu peux t'avancer dans le cours de JavaScript de CodeCademy, qui te permettra d'aller plus en profondeur sur les intéractions avec JavaScript.

Projets

1. Introduction

Dans ce projet, tu vas ajouter du JavaScript à ton site BetterMotherFuckingWebsite afin d'afficher une superbe popup qui propose à notre utilisateur d'aller sur plein de sites internet.

2. Le contenu

2.1. La popup trop cool

Ton site BetterMotherFuckingWebsite est trop bien, mais une grande entreprise de vidéos pornographiques te propose un contrat que tu ne peux pas refuser : contre 250 milliards de dollars zimbabwéen, tu dois afficher une popup en plein milieu de la page qui affiche "Hey le site xxxvidsxxx est trop bien. Viens dessus stp please" au bout de 10 secondes sur le site. La popup peut être en alert().

Ta mission est d'implémenter sur le site BetterMotherFuckingWebsite cette popup.

2.2. Pimp de la popup

Maintenant, nous allons améliorer la popup pour laisser à l'utilisateur le choix ce ce qu'il fera. La popup va demander à l'utilisateur s'il veut aller sur xxxvidsxxx. Si l'utilisateur répond par l'affirmative, le site devra le rediriger sur cette url : https://www.youtube.com/watch?v=dQw4w9WgXcQ. S'il répond par la négative, rien ne se passe.

Nous te laissons le choix du type de popup à afficher, ainsi que le format de la question.

3. Pour aller plus loin

Le mode psychédélique

Ton site BetterMotherFuckingWebsite déchire, mais plein de gens se plaignent de son manque de vie. Nous allons donc lui donner de un peu peu de peps.

Créé un bouton qui déclenche le "Mode Psychédélique" : quand un utilisateur clique sur le bouton, le background de la page devra changer en magenta.

Puisque c'est toi, nous allons te proposer d'autres fonctionnalités. Le bouton devra aussi :

  • Changer la couleur des polices des paragraphes en blue
  • Changer la couleur des polices de tous les headings en green
  • Changer la police des paragraphes en papyrus et la police des headers en Comic Sans

4. Rendu attendu

Voici ce que nous attendons comme rendu pour le projet :

  • Un lien vers un repository GitHub qui contient le site
  • Ce site devra être BetterMotherFuckingWebsite, avec une popup JavaScript qui demande si l'utilisateur veut aller sur xxxvidsxxx
  • Évidemment, le fichier README.md devra présenter le projet, et donner un lien en production
